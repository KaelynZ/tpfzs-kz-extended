:: Eli's Main Menu Replacer ::

Mod changes the background image and music for the Skyrim Main Menu. Also removes the fog.

I was just testing. I found this AMAZING artist, Vindsvept, through Hodilton videos and had to use this song. 
I made it for myself, thought I would share. 

:: INSTALLATION ::

Extract the rar and place Data folder to your .....\Steam\SteamApps\common\Skyrim, merge and overwrite when prompted.

:: UNINSTALLATION ::

Remove files from 
..\Steam\SteamApps\common\Skyrim\Data\meshes\interface\logo <- delete logo.nif
..\Steam\SteamApps\common\Skyrim\Data\Textures\interface\objects <- delete mainmenuwallpaper.dds
..\Steam\SteamApps\common\Skyrim\Data\Music\special <- delete mus_maintheme.xwm
